# AI Invoice Auditor

A modern web application prototype for an AI Invoice Auditor system—a decision-support tool designed to help finance teams identify potential discrepancies in vendor invoices.

## Features

- **Hero Section** — Full-screen landing with particle animation
- **Problem Section** — Explains why invoice auditing is difficult
- **How It Works** — Step-by-step process explanation
- **What It Checks** — Discrepancy types (GST, HSN, duplicates, etc.)
- **Demo Interface** — Upload invoices, view extracted data, flagged discrepancies
- **Dashboard** — Mock analytics with charts
- **Design Philosophy** — Product reasoning
- **Write-Up** — 500-word insight explanation
- **Footer** — Links and disclaimer

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** TailwindCSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Charts:** Recharts

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design

- **Colors:** Beige (#F5F1E8), Navy (#0F1F3D), White
- **Style:** Minimal, elegant, product-focused
- **Font:** San Francisco Pro (system fallback)

This tool highlights potential discrepancies and should be used as a decision-support system.
